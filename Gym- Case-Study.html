Customer Feedback System for Gym

Designed a lightweight digital feedback collection system to help a local gym gather and analyze member feedback for continuous service improvement.

📌 Project Overview

This project involved designing a no-code feedback pipeline using Google Forms, Sheets, Excel, Trello, and Jira. I implemented Agile planning and designed a full workflow from user input to dashboard insights.

🔧 Tools Used

  • Google Forms – Feedback collection
  • Google Sheets – Real-time data capture
  • Excel – Dashboard with charts and trends
  • Jira – Epics, Stories, and Tasks management
  • Trello – Internal operations tracker

🧩 Project Structure

The Jira board was structured into an Epic with these Stories:

  • Story 1: Set up and test a Google Form with rating and comment fields
  • Story 2: Auto-link form to Google Sheets and clean data
  • Story 3: Build dashboard in Excel with graphs
  • Story 4: Create process map and Trello board
  • Story 5: Prepare portfolio documentation

📈 Key Outcomes

  • Simplified feedback capture process
  • Visual dashboard for quick insight into common complaints and satisfaction trends
  • Improved potential response time to complaints
  • Ready-to-deploy template for similar small businesses

📂 Deliverables

  • Google Form (Live Preview)
  • Google Sheet + Excel Dashboard
  • Trello Board (Read-only)
  • Jira Board

💡 Lessons Learned

This case sharpened my Agile thinking and systems analysis skills. It taught me how to build lightweight solutions and turn raw data into actionable insights — without code.
